# 测风数据邮件日报监测工具说明书

作者：楚煜

## 1. 软件用途

测风数据邮件日报监测工具用于从 163/188 邮箱读取测风数据邮件，按配置规则筛选邮件和附件，保存附件记录，生成测风日报，并可选择通过邮件发送日报。

软件适合以下场景：

- 每天检查测风数据邮件是否按时到达。
- 自动筛选指定发件人、主题关键词和附件类型。
- 识别小文件、缺失数据、连续缺失等异常情况。
- 生成本地 Excel/HTML 日报。
- 将日报发送给指定收件人和抄送人。

## 2. 文件说明

便携版目录中主要文件和文件夹如下：

| 名称 | 说明 |
| --- | --- |
| `WindMailMonitor.exe` | 软件主程序，双击即可打开 |
| `config/config.yaml` | 当前本地配置文件，会保存邮箱账号、授权码、规则和收件人 |
| `config/config.example.yaml` | 配置模板 |
| `data/` | 附件和中间数据保存目录 |
| `reports/` | 生成的日报保存目录 |
| `logs/` | 运行日志目录 |
| `database/` | 本地数据库目录 |
| `_internal/` | 程序运行依赖文件，请勿删除 |

## 3. 启动软件

1. 解压 `WindMailMonitor-portable.zip`。
2. 进入 `WindMailMonitor-portable` 文件夹。
3. 双击 `WindMailMonitor.exe`。
4. 打开后会看到三个页面：`邮箱设置`、`日报规则`、`运行`。

注意：请保持 `_internal` 文件夹和 `WindMailMonitor.exe` 在同一目录下，否则软件可能无法启动。

## 4. 首次使用配置

### 4.1 邮箱设置

进入 `邮箱设置` 页面，填写以下内容：

| 字段 | 说明 |
| --- | --- |
| 163 邮箱账号 | 用于接收测风数据邮件的邮箱账号，例如 `example@163.com` |
| 客户端授权码 | 163/188 邮箱的客户端授权码，不是网页登录密码 |
| IMAP 服务器 | 默认 `imap.163.com`，通常不用改 |
| SMTP 服务器 | 默认 `smtp.163.com`，通常不用改 |
| 日报接收人 | 日报发送对象，多个邮箱用英文逗号分隔 |
| 抄送 | 抄送对象，多个邮箱用英文逗号分隔 |
| 运行完成后发送日报邮件 | 勾选后会自动发送日报；取消勾选则只生成本地日报 |

填写完成后点击右上角 `保存设置`。

### 4.2 获取 163/188 邮箱客户端授权码

一般步骤如下：

1. 登录 163 或 188 邮箱网页版。
2. 进入邮箱设置。
3. 开启 `IMAP/SMTP` 服务。
4. 按邮箱页面提示生成客户端授权码。
5. 将授权码填写到软件的 `客户端授权码` 字段。

注意：客户端授权码与网页登录密码不同。请不要把授权码发给无关人员。

## 5. 日报规则配置

进入 `日报规则` 页面，配置筛选和提醒规则。

### 5.1 规则参数

| 字段 | 说明 |
| --- | --- |
| 小文件阈值 KB | 附件小于该大小时，会被标记为小文件风险 |
| 连续缺失提醒天数 | 测风数据连续缺失达到该天数时，会触发提醒 |
| 附件保存目录 | 邮件附件保存位置，默认 `./data` |
| 日报保存目录 | 日报输出位置，默认 `./reports` |

### 5.2 邮件筛选规则

| 字段 | 说明 |
| --- | --- |
| 允许发件人 | 每行一个邮箱地址。填写后，只处理这些发件人的邮件 |
| 主题关键词 | 每行一个关键词。填写后，只处理主题包含这些关键词的邮件 |

默认主题关键词包括：

- 测风
- 测风数据
- 风速数据
- Molas B300
- HN-
- GX-

默认附件类型包括：

- `.rld`
- `.zip`
- `.txt`
- `.csv`
- `.xls`
- `.xlsx`
- `.rar`

配置完成后点击右上角 `保存设置`。

## 6. 生成日报

进入 `运行` 页面：

1. 在 `统计日期` 中填写要统计的日期，格式为 `YYYY-MM-DD`，例如 `2026-07-08`。
2. 如需连接邮箱读取邮件，不要勾选 `不连接邮箱，仅根据数据库重新生成日报`。
3. 如只想基于已有数据库重新生成日报，可勾选该选项。
4. 点击 `立即生成日报`。
5. 运行过程会显示在 `运行日志` 中。
6. 生成完成后，可点击右上角 `打开报告目录` 查看日报文件。

## 7. 常用操作

### 7.1 只生成日报，不发送邮件

在 `邮箱设置` 页面取消勾选 `运行完成后发送日报邮件`，然后保存设置。之后运行时只会在本地生成日报，不会发送邮件。

### 7.2 重新生成某一天日报

进入 `运行` 页面，填写对应统计日期。如果不需要重新读取邮箱，可以勾选 `不连接邮箱，仅根据数据库重新生成日报`。

### 7.3 查看生成结果

点击右上角 `打开报告目录`，或手动进入 `reports/` 文件夹。

### 7.4 查看运行日志

运行界面会显示本次运行日志；历史日志文件保存在 `logs/` 文件夹。

## 8. 数据安全说明

请重点注意以下内容：

- `config/config.yaml` 会保存邮箱账号和客户端授权码。
- 不要把 `config/config.yaml` 上传到 GitHub、网盘或发送给无关人员。
- 不要随意删除 `database/`，否则历史邮件记录和日报判断依据可能丢失。
- 不要删除 `_internal/`，否则软件无法正常启动。
- 如果需要把软件发给别人，建议先清空 `config/config.yaml` 中的邮箱账号和授权码。

## 9. 常见问题

### 9.1 双击 exe 没反应

请检查：

- 是否完整解压了压缩包。
- `_internal` 文件夹是否还在。
- 杀毒软件是否拦截了程序。
- 是否从压缩包内部直接运行，建议先完整解压后再运行。

### 9.2 邮箱登录失败

请检查：

- 邮箱账号是否填写完整。
- 填写的是客户端授权码，而不是网页登录密码。
- 163/188 邮箱是否已开启 IMAP/SMTP 服务。
- 网络是否能连接邮箱服务器。

### 9.3 没有筛选到邮件

请检查：

- 统计日期是否正确。
- 发件人是否在 `允许发件人` 列表中。
- 邮件主题是否包含配置的关键词。
- 邮件是否带有支持的附件类型。
- 邮件是否在收件箱中。

### 9.4 日报没有发送

请检查：

- `运行完成后发送日报邮件` 是否已勾选。
- 日报接收人是否填写正确。
- SMTP 服务器是否为 `smtp.163.com`。
- 客户端授权码是否有效。

### 9.5 想预览日报但不发邮件

取消勾选 `运行完成后发送日报邮件`，保存设置后再运行。

## 10. 建议使用流程

日常使用建议按以下步骤：

1. 打开软件。
2. 检查邮箱设置和日报收件人。
3. 检查日报规则是否需要调整。
4. 进入运行页，确认统计日期。
5. 点击 `立即生成日报`。
6. 查看运行日志，确认是否成功。
7. 打开报告目录检查生成的日报。

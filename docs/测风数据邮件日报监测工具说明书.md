# 测风数据邮件日报监测工具客户使用手册

软件名称：测风数据邮件日报监测工具  
当前版本：V1.4.0  
作者：楚煜  
邮箱：15274958341@163.com  

## 一、软件用途

本软件用于检查测风数据邮件是否正常到达，自动保存附件，识别缺测、连续缺测、小文件异常和未识别附件，并生成 Excel 与 HTML 日报。

适用场景：

- 每天检查测风塔或测风雷达数据是否到达。
- 根据发件人、邮件主题关键词和附件名称筛选测风数据。
- 统计指定日期的测风数据接收情况。
- 自动生成日报文件，便于发送、归档和核查。
- 在授权范围内离线使用软件。

## 二、软件目录说明

解压后的软件目录一般包含以下内容：

| 名称 | 说明 |
| --- | --- |
| `WindMailMonitor.exe` | 主程序，双击打开 |
| `_internal/` | 程序运行依赖文件，请不要删除 |
| `config/` | 配置文件目录 |
| `data/` | 邮件附件保存目录 |
| `database/` | 本地数据库目录 |
| `logs/` | 运行日志目录 |
| `reports/` | 日报输出目录 |
| `docs/` | 软件说明和授权相关文档 |
| `DELIVERY_MANIFEST.json` | 交付清单 |

请不要只单独复制 `WindMailMonitor.exe`。移动软件时，应整体移动整个文件夹。

## 三、首次使用流程

1. 解压软件压缩包。
2. 打开解压后的文件夹。
3. 双击 `WindMailMonitor.exe`。
4. 首次使用如果未授权，软件会进入“授权”页面。
5. 完成授权后，依次配置“邮箱设置”和“日报规则”。
6. 进入“运行”页面，选择统计日期并点击“立即生成日报”。
7. 点击右上角“打开报告目录”查看生成结果。

## 四、离线授权使用

### 4.1 导出授权申请

首次使用或换电脑时，需要先导出授权申请：

1. 打开软件，进入“授权”页面。
2. 填写客户名称。
3. 确认页面显示“短设备编号”和“完整设备码”。
4. 点击“导出授权申请”。
5. 选择保存目录，生成 `.req` 授权申请文件。
6. 将 `.req` 文件发送给授权签发人员。

授权申请文件不包含邮箱密码、邮箱客户端授权码、业务数据或许可证签名。

### 4.2 导入许可证

收到 `.lic` 许可证文件后：

1. 打开软件，进入“授权”页面。
2. 点击“导入许可证”。
3. 选择签发人员提供的 `.lic` 文件。
4. 软件提示导入成功后，即可正常使用。

许可证与当前电脑绑定。如果更换电脑，需要重新导出授权申请并重新签发许可证。

### 4.3 续期或升级

许可证即将到期或需要升级授权时：

1. 进入“授权”页面。
2. 点击“导出续期申请”或“导出升级申请”。
3. 将生成的 `.req` 文件发送给授权签发人员。
4. 收到新的 `.lic` 文件后，点击“导入许可证”完成更新。

## 五、邮箱设置

进入“邮箱设置”页面，填写邮箱和日报发送信息。

| 字段 | 说明 |
| --- | --- |
| 邮箱账号 | 用于接收测风数据邮件的邮箱，例如 `example@qq.com`、`example@163.com`、`example@gmail.com` |
| 客户端授权码/应用专用密码 | 邮箱客户端授权码或应用专用密码，不是网页登录密码 |
| IMAP 服务器 | 填写邮箱账号后软件会自动识别 QQ、163、Gmail、Outlook、126、Foxmail 的 IMAP 服务器 |
| SMTP 服务器 | 填写邮箱账号后软件会自动识别 QQ、163、Gmail、Outlook、126、Foxmail 的 SMTP 服务器 |
| 日报接收人 | 接收日报的邮箱，多个邮箱用英文逗号分隔 |
| 抄送 | 抄送邮箱，多个邮箱用英文逗号分隔 |
| 运行完成后发送日报邮件 | 勾选后自动发送日报；取消勾选则只生成本地日报 |

填写完成后，点击右上角“保存设置”。

### 5.1 客户端授权码说明

客户端授权码不是邮箱网页登录密码。一般需要在邮箱网页版设置中开启 IMAP/SMTP 服务后生成。

常见步骤：

1. 登录邮箱网页版。
2. 进入邮箱设置。
3. 开启 IMAP/SMTP 服务。
4. 按页面提示生成客户端授权码。
5. 将授权码填入软件“客户端授权码”字段。

## 六、日报规则设置

进入“日报规则”页面，配置检查规则和邮件筛选规则。

### 6.1 规则参数

| 字段 | 说明 |
| --- | --- |
| 小文件阈值 KB | 附件小于该大小时，标记为文件大小异常；同时，附件低于同一测风塔历史正常附件平均大小的 80% 时也会标记异常 |
| 连续缺失提醒天数 | 连续缺失达到该天数时，进入连续缺失提醒 |
| 附件保存目录 | 附件保存位置，默认 `./data` |
| 日报保存目录 | 日报保存位置，默认 `./reports` |

### 6.2 邮件筛选规则

| 字段 | 说明 |
| --- | --- |
| 允许发件人 | 每行一个邮箱地址。示例：`******@***.com` |
| 主题关键词 | 每行一个关键词。示例：`塔号`、`邮箱主题关键词` |
| 无效塔号 | 每行一个塔号。适用于倒塔、拆除或永久停用后不再需要监测的测风塔 |

说明：

- 允许发件人填写后，软件只处理这些发件人发来的邮件。
- 主题关键词填写后，软件优先处理主题包含关键词的邮件。
- 无效塔号填写后，软件不再保存该塔号的新附件；已有历史记录也不会参与接收、缺测、连续缺测或异常提醒。
- 塔号可填写带前导零或不带前导零的形式，例如 `009357` 与 `9357` 会识别为同一塔号。
- 如果附件名称前 6 位都是数字，软件也会认为它是测风数据附件。
- 默认示例 `******@***.com`、`塔号`、`邮箱主题关键词` 只是占位提示，不会作为实际过滤条件。

## 七、生成日报

进入“运行”页面后：

1. 在“统计日期”中填写日期，格式为 `YYYY-MM-DD`。
2. 如果需要读取邮箱，不要勾选“不连接邮箱，仅根据数据库重新生成日报”。
3. 如果只想用已有数据库重新生成日报，可以勾选“不连接邮箱，仅根据数据库重新生成日报”。
4. 点击“立即生成日报”。
5. 运行过程会显示在“运行日志”中。
6. 生成完成后，点击“打开报告目录”查看日报。

生成结果包括：

- Excel 日报：`.xlsx`
- HTML 日报：`.html`
- 邮件附件：保存到 `data/`
- 运行日志：保存到 `logs/`

## 八、日报内容说明

Excel 日报通常包含以下工作表：

| 工作表 | 说明 |
| --- | --- |
| 总体汇总 | 当天接收、缺失、异常、未识别数量汇总 |
| 今日接收清单 | 已识别测风塔或测风雷达的接收情况 |
| 缺失提醒 | 当天未收到但历史上出现过的测风塔 |
| 连续缺失提醒 | 连续多天未收到的测风塔 |
| 文件大小异常 | 空文件、小于固定阈值，或低于同塔历史正常附件平均大小 80% 的附件 |
| 邮件附件明细 | 邮件和附件明细 |
| 未识别附件 | 未识别为测风数据的附件 |

## 九、缺测判断逻辑

软件会根据历史已出现过的塔号建立“应到清单”，再与统计日期当天实际收到的数据进行比较。

举例：

- 7 月 11 日收到过 `0010` 测风塔数据。
- 7 月 12 日没有收到 `0010` 测风塔数据。
- 统计日期选择 `2026-07-12` 时，软件会将 `0010` 标记为缺测。

这种逻辑不要求先生成前一天日报。只要数据库中已有历史接收记录，第一次统计当天就能显示缺测结果。

### 支持的邮箱文件夹和附件格式

- 软件同时扫描“收件箱”和邮箱服务器标记的“已发送”文件夹。
- 支持 `.rld`、`.swift`、`.RWD`、`.dat`、`.zip`、`.txt` 六种测风数据附件。
- 同一附件同时出现在收件箱和已发送文件夹时，软件按文件内容复用记录，不重复统计。

## 十、重复数据处理

如果同一天重复运行，软件会自动处理重复附件：

- 同名且内容一致的附件会复用原文件，不再生成重复文件。
- 报表导出时，会按日期、塔号、文件名、文件大小、发件人和主题进行去重。
- 因此多次运行同一统计日期，不应在 Excel 中出现重复明细。

## 十一、异常图标提醒

如果日报结果中存在以下情况，软件图标会显示红色感叹号：

- 今日缺失提醒
- 连续缺失提醒
- 文件大小异常
- 未识别附件
- 上次运行失败

再次运行后如果异常消失，图标会恢复为普通信封样式。

## 十二、常见问题

### 12.1 提示邮箱登录失败

常见原因：

- 邮箱账号填写不完整。
- 客户端授权码填写错误。
- 将网页登录密码误填为客户端授权码。
- 邮箱未开启 IMAP/SMTP 服务。

处理方法：

1. 登录邮箱网页版。
2. 确认 IMAP/SMTP 服务已开启。
3. 重新生成客户端授权码。
4. 回到软件保存设置后重新运行。

### 12.2 提示 SMTPUTF8 错误

常见原因是收件人、抄送人或发件邮箱中包含中文、空格或特殊字符。

处理方法：

- 确认收件人和抄送人均为标准英文邮箱地址。
- 多个邮箱之间使用英文逗号分隔。
- 删除邮箱前后的空格。

### 12.3 没有统计出某个附件

请检查：

- 发件人是否被“允许发件人”过滤。
- 邮件主题是否包含配置的主题关键词。
- 附件扩展名是否在支持范围内。
- 附件名称是否能识别塔号。
- 统计日期是否与附件日期或邮件内容日期一致。

### 12.4 Excel 中显示未识别附件

说明软件保存了附件，但没有识别出塔号或测风数据日期。

可检查：

- 附件名称是否包含塔号。
- 邮件主题是否包含塔号。
- 附件内容是否包含日期。
- 是否为非测风数据附件。

### 12.5 授权显示设备不匹配

许可证绑定电脑。出现该提示通常表示：

- 换了电脑。
- 系统硬件标识发生明显变化。
- 使用了其他电脑签发的许可证。

处理方法：重新导出授权申请，交由签发人员重新签发。

## 十三、使用建议

- 每次修改邮箱、规则或日报收件人后，请点击“保存设置”。
- 建议每天固定时间手动运行一次。
- 如果只想重新生成报告，不想重新读取邮箱，可以勾选“不连接邮箱，仅根据数据库重新生成日报”。
- 不要删除 `_internal/`、`database/`、`config/` 等目录。
- 迁移软件时，请整体复制整个软件文件夹。
